dkim_sigkey_t

DESCRIPTION
Purpose A dkim_sigkey_t points to the PEM-formatted private key to be used to sign messages. (For now, it is simply a pointer to a string of type unsigned char.)
NOTES
  • The calling application should read the PEM file into memory and pass it in as this data type. Carriage return and newline characters (ASCII 13 and 10, respectively) need not be stripped.
