dkim_dns_set_query_start()

[back to index]

SYNOPSIS 
#include <dkim.h>
void *dkim_dns_set_query_start(
        int (*func)(void *srv,
	            int type,
	            char *query,
	            unsigned char *buf,
	            size_t buflen,
	            void **qh);
);
Specifies a function that the library should call when it wishes to initiate a DNS query.
DESCRIPTION
Called When dkim_dns_set_query_start() can be called at any time prior to DNS operations while verifying.
ARGUMENTS
ArgumentDescription
func A pointer to a function that should be called whenever libopendkim wishes to initiate a DNS query. The function should expect the following arguments:
ArgumentDescription
srv Pointer to the DNS service handle previously provided in a call to dkim_dns_set_query_service().
type The DNS resource record (RR) type being requested.
query The name of the DNS record being requested.
buf Pointer to a buffer that should receive the reply.
buflen Number of bytes available at buf.
qh Pointer to a (void *) that will be updated to contain a handle referencing the new query.
RETURN VALUES
  • 0 on success.
  • Non-zero on error.
NOTES
  • None.
