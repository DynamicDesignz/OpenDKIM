dkim_dns_set_query_service()

[back to index]

SYNOPSIS 
#include <dkim.h>
void *dkim_dns_set_query_service(
        void *ptr);
);
Specifies an opaque context pointer identifying a DNS resolver service to be used by the library. Returns any previous pointer.
DESCRIPTION
Called When dkim_dns_set_query_service() can be called at any time prior to DNS operations while verifying.
ARGUMENTS
ArgumentDescription
ptr A user-provided pointer to a DNS query service handle that will be passed to other DNS query functions provided by the user.
RETURN VALUES
  • Always returns the previous pointer, or NULL if there was none.
NOTES
  • None.
