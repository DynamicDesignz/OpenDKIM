dkim_dns_set_query_cancel()

[back to index]

SYNOPSIS 
#include <dkim.h>
void *dkim_dns_set_query_cancel(
        int (*func)(void *srv,
                    void *qh);
);
Specifies a function that the library should call when it wishes to cancel a pending DNS query.
DESCRIPTION
Called When dkim_dns_set_query_cancel() can be called at any time prior to DNS operations while verifying.
ARGUMENTS
ArgumentDescription
func A pointer to a function that should be called whenever libopendkim wishes to cancel a pending DNS query. The function should receive two void pointer arguments: one will be the one provided via a prior call to dkim_dns_set_query_service() identifying the DNS service being used, and the other will be returned as part of a prior call to the function named by dkim_dns_set_query_start() identifying a specific query.
RETURN VALUES
  • None.
